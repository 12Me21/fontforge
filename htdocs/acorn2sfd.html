acorn2sfd
A program for comparing fonts

acorn2sfd [-version] [-help] {acorndir ...}

acorn2sfd will take several Acorn RISC/OS font directories and convert them into PfaEdit's sfd files.

An Acorn RISC/OS font directory should contain two files, one named Outlines and one named Intmetrics (the Intmetrics file may be omitted, in which case the advance widths will not be known). acorn2sfd will read both files and generate an sfd file from them (which PfaEdit can turn into TrueType or Postscript).

-help will provide a mini description and will list the available options.

-usage will list the available options.

-version will display the current version of acorn2sfd (a six digit string containing the date stamp of the source files).

Bugs

  • Acorn fonts usually do not contain encoding data so acorn2sfd assumes iso-latin1 (or unicode for big fonts).
  • Acorn hinting data are lost.
  • Acorn kerning data are currently lost.
