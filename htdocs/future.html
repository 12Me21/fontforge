Currently under development

  • A command that will allow you to
    • Make a character/font bolder
    • Extend or Condense a character/font
    • Change the X-Height of a font
    • Regularize stem widths
  • Some vague support for CID keyed opentype fonts
  • A Find/Replace command that will allow you to search for features and replace them with others
    The dlg will provide two little drawing panes (one to find, one to replace) the program will search for a series of points in the same relative positions (possibly translated, flipped or followed backwards) and if found will replace it with the the points in the replace window. I see two uses for this:
    • Change all the serifs in a font from one style to another.
    • Search for all inline copies of a glyph and replace each with a reference.

Musings

  • Some kind of support for multiple master fonts
  • An editing mode which supports more features that type3 fonts could use (ie. multiple colors, strokes, ...)
  • hinting of italic serifs in true type output
  • proper construction of arabic (right to left) ligatures
  • A scroll bar in the metrics view
  • Better import of xfig files (a real X-Spline -> Bezier spline converter)
  • Mechanism for creating fonts with multiple instances of the same character, each slightly different, coupled with a postscript routine which will for each character in a string, pick a random version of that character to display. An attempt to yield a slight non-uniformity to the font which may make its output look more interesting.
