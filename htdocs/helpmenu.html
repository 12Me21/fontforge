Help Menu

The Help menu has only two entries:

  • Help
  • About
Help
Invokes a browser and brings up the pfaedit web site.

If you define the environment variable BROWSER so that it contains the name of the browser you like then PfaEdit will use that, otherwise it will try to guess something appropriate by looking at your PATH.

PfaEdit will search for help in either /usr/share/doc/pfaedit/ or on http://pfaedit.sf.net/.

About
Will display the splash screen again.
